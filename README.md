# Тетрис

Курсовой проект по дисциплине «Программирование (часть 2)»  
Направление: 09.03.02 — Информационные системы и технологии  
Томский политехнический университет, 2025

## Описание
Классическая игра **Тетрис**, реализованная на Python с использованием библиотеки **Pygame**.  
Поддерживает:
- Разные размеры поля (10×20, 12×24, 15×30)
- Разные размеры фигур: тетромино, пентамино, гексамино, гептамино
- Сохранение и загрузку игры
- Таблицу лидеров с фильтрацией
- Настраиваемое управление через `config.ini`
- Звуковые эффекты и фоновую музыку
- "Призрачную" фигуру-подсказку

## Установка
1. Скачайте установочный файл `Tetris_Setup.exe`
2. Запустите и следуйте инструкциям мастера установки
3. Запустите игру через ярлык или исполняемый файл

## Управление
- **← →** — переместить фигуру
- **↑** — повернуть фигуру
- **↓** — ускоренное падение
- **Пробел** — мгновенное падение
- **P** — пауза
- **S** — сохранить игру
- **N** — новая игра
- **Esc** — вернуться в меню

Настройки клавиш можно изменить в файле `config.ini`.
